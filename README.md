# Python File Organizer by Name

This program is extremely simple, it allows you to select a file directory, and organize on the files inside it into folders based on their naming convention.

## Instructions
1. Download the .exe
2. When in the .exe, select the directory that you want to organize
3. Choose the seperator, for example if the folder contains: A1-14e412, A1-55555, A2-444412, you can select - as the seperator, and you will get two folders named A1 and A2.
4. Click organize, and you should be done.
